Week 11: Lab #21

Email Validation

DUE: Thursday, 3/30 @ 5pm



  • Create a form page that asks someone to fill in their email address into a text box. The page should have a form, a text input field, and a button that says "Validate email".
  • When the user presses the "Validate email" button, it should execute a JavaScript routine that checks the value of the text input field to make sure the data has both an at (@) character and a period (.) character, and that the period comes after the @ in the string.
  • If the email address is valid, it should trigger an alert that says "Email address is valid", if not it should trigger an alert that says "Please enter a valid email address".

Upload your completed lab by the due date specified above and make sure that you add a working link to it on your class page, otherwise it will not be reviewed and you will not receive credit.
