Week 2: Lab #04

Floating Div Test

DUE: Tuesday, 1/24 @ 5pm



Create a web page with a minimum of 10 different divs, all of them set to float, in order to see how they re-arrange while adjusting the browser width. The divs can be varying widths and heights, and make sure you style the divs with different background colors and foreground colors to create some obvious visual differentiation between them.
