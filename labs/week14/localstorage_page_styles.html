Week 14: Lab #27

localStorage Page Styles

DUE: Thursday, 4/20 @ 5pm



  • Add three drop-down form elements:
    1. One for foreground color options (minimum of 5).
    2. One for background color options (minimum of 5).
    3. One for a welcome message (minimum of 5).
  • Add a form button that has an event listener so when pressed it will save the foreground color, background color, and welcome message as localStorage variables.
  • When the page is reloaded, it should check to see if localStorage variables exist, and if so apply the stored variables of the foreground and background colors to the appropriate CSS elements and the welcome message to the appropriate HTML content.
