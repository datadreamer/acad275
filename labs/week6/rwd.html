Week 6: Lab #12

Responsive Design Layout

DUE: Tuesday, 2/21 @ 5pm



Create a webpage that uses responsive web design techniques, either using percentage based fluid layouts or using flexbox techniques, as well as media queries to allow for the layout to adjust to different dimensions, creating a desktop layout and a mobile layout. The page can be about any subject matter you choose and use any content you derive from wherever you like (do try to keep it class appropriate). The page must have the following criteria:

  • The page must fill the entire browser width, with no margins around the body.
  • The page must have a header area that spans the full width of the browser with a title of some type inside it.
  • The page must have a sidebar on the left or right that has a fixed width.
  • The page must have a main content area with a variable width that adjusts with browser dimensions.
  • The main content area must have a gallery of thumbnail images that wrap when the browser width is adjusted.
  • The page must have a footer area that spans the full width of the browser.

Upload your completed lab by the due date specified above and make sure that you add a working link to it on your class page, otherwise it will not be reviewed and you will not receive credit.
