Week 7: Lab #13

JavaScript Color Setter

DUE: Thursday, 2/23 @ 5pm



  • Create a page about a hobby or interest you have with a two column layout: a sidebar (left or right) and the main section.
  • The main section should describe some of your hobbies and contain at least two headlines, five paragraphs of text, and two or more images.
  • The sidebar should also contain two sections:
    • The sections should be labeled "Text" and "Background".
    • Each section should have 10 radio buttons with different colors.
    • Users should be able to select only one radio button per column at a time.
    • In a script block, write vanilla JavaScript (no jQuery) that binds a listener to each radio button for the click event, and then changes the content div to the corresponding background or text color.
  • A "reset" button near the color selections that resets the pages colors to their starting values.

Upload your completed lab by the due date specified above and make sure that you add a working link to it on your class page, otherwise it will not be reviewed and you will not receive credit.
