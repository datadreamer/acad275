Adding a columns property will automatically divide the text evenly within the HTML element. For compatibility reasons, you must also include a -webkit-columns and -moz-columns properties to ensure compatibility across different types of browsers.

.col2{
    columns: 2;
    -webkit-columns: 2;
    -moz-columns: 2;
}

Rules can automatically be added between columns and the size of the gap can be adjusted using column-rule and column-gap.

.fatgap{
    columns: 3;
    -webkit-columns: 3;
    -moz-columns: 3;
    column-rule: white 2px solid;
    column-gap: 50px;
}
